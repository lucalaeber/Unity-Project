Ninja Showdown

Ninja Showdown is an action-platformer prototype where the player controls a ninja navigating through levels filled with obstacles, enemies, and collectibles.
This README explains how to install, run, and play both the keyboard and controller versions of the game.

⸻

System Requirements
	
	•	Operating System: Any (Windows, macOS, Linux)
	
	•	Processor: Any
	
	•	Memory: Any
	
	•	Graphics: Any GPU supported by Unity
	
	•	DirectX: Any
	
	•	Storage: ~500 MB free

⸻

Installation

1. Downloading the Game
	
	•	Download the game files from the repository or provided link.
	
	•	Extract the ZIP file to any location.

2. Dependencies
Ensure the following are installed:
	
	•	Unity Player
	
	•	Microsoft Visual C++ Redistributable (Windows)
	
	•	Mono Framework (macOS)

⸻

Controls

Keyboard Controls
	
	•	Move Left: A or Left Arrow
	
	•	Move Right: D or Right Arrow
	
	•	Jump: Space
	
	•	Double Jump: Space (while in air)
	
	•	Dash: Double-tap A or D
	
	•	Pause: Esc

Controller Controls
	
	•	Move Left/Right: Left Analog Stick or D-Pad
	
	•	Jump: A (Xbox) / X (PlayStation)
	
	•	Double Jump: A/X while in air
	
	•	Dash: B (Xbox) / O (PlayStation)
	
	•	Pause: Start/Options

⸻

Known Issues
	
	•	Limited feedback when completing a level.
	
	•	Controller input may not work properly in the main menu.
	
	•	Camera may lose track of the player when jumping too high.
	
	•	Score screen may fail to update top scores correctly.
	
	•	Minor lag may occur when playing through a browser URL.

Most of these issues are explained in detail in the project report.

⸻

How to Import the Unity Package
	
	1.	Open Unity Hub and create a new 3D project (or open an existing one).
	
	2.	Once the project loads, go to the top menu: Assets → Import Package → Custom Package…
	
	3.	Select the file: HCI_Group39.unitypackage
	
	4.	A window will appear showing all assets included in the package.
	
	5.	Keep all items selected and click Import.
	
	6.	After the import finishes, open the main scene (usually found under Assets/Scenes/ or a similar folder).
	
	7.	Press Play to start the game.
